Introduction

Literature Mapper is a plugin for QGIS that inserts geoJSON geometry strings into a Zotero database to facilitate the mapping of study site locations. My Literature Map is an example of a map that has benefitted from this tool.

Credits: Literature Mapper is developed by Michele Tobias and Alex Mandel.

Documentation

Download & Install

Download Literature Mapper at the GitHub repository for the project and place it in your QGIS plugins folder. When you start QGIS, it should be available on the Installed Plugins menu of the Manage and Install Plugins tool (find this in the Plugins Menu).
